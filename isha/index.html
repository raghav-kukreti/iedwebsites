About

For our mini project we were required to create an interactive lamp from scratch. The objective of this project was to put everything we had learnt in class to practical use so that we could build a prototype which could actually be sold in the market. It was necessary for us to create something which was fully functional without any glitches and aesthetically good-looking to appeal to the customer.

We had to build it using arduino uno which is an open source micro‐controller.Some of the things that we were also required to incorporate in our project were a bluetooth module, a motor, different sensors and LEDs. We also connected our lamp to an app which we created to make it more user-friendly.

Implementation

Basic Idea

Our group decided to make a conventional lamp system where the base would contain all the machinery and the upper canopy contained LED lights that were connected in parallel. The canopy would be connected to the base through a pipe which contained 2 wires to power the LEDs.

The lamp would have the following features :

  1. App controlled functionality.
  2. Pivoted left and right movement using a motor which would be controlled by the app and a touch sensor.
  3. Blinking of LEDs on sensing notifications received on the phone.
  4. ON/OFF switching using an ultrasonic sensor.
  5. Intensity of the LEDs controlled bu the distance of the hand from the lammp using LDR.

Process

We started by making the circuit by connecting the different components such as the bluetooth module, LDR, ultrasonic sensor etc. using wires on the breadboard so that it could be linked with the arduino. Codes were written for each component which would later be loaded on the arduino so that all the components could work using an arduino.

We then moved on to make the structure. First the canopy was made which had a cardboard base and top and they were connected by 6 ice- cream sticks, each containing an LED which were connected in parallel. The canopy was then covered with butter paper to complete it. Next we made the base which was a rectangular box containing all the machinery which was powered by a 9V battery. The canopy and base were connected by a rectangular cardboard pipe which took wires from the LEDs to the circuit. Both the pipe and base were covered with chart paper to beautify the lamp.

Finally we made the app to control the lamp using MIT App Inventor 2. It connected our phone to the lamp via bluetooth. We made the app in such a way that the lamp could be switched on or off using it and aslo be moved. It was through this app that the lamp could also detect notifications on our phone.


Final Look

When we were finished, we had a functional lamp with all the features that we had planned to incorporate. We had a lamp which moved towards the user if needed to provide more light. The lamp also lit up if some books or any object was placed in front of it. It also informed us by blinking if we had received any notifications on our phone.

Reflections

Being first year college students, we could not acheive the finess we hoped for. We could not hide our sensors as properly as they shouls have been hidden. We had also hoped to make the LED blink different colours of light according to the social media platform our phone received notifications from. (example ‐ green for whatsapp and blue for messenger) We also originally wanted to change the height of our lamp by upward-downward movement using a stepper motor but due to lack of availability of the required materials in such a short period of time we could not acheive that.

Despite all the challenges we faced, this project taught me a lot and exposed me to a wide variety of new things. This was the first time that any of us had ever made such a machine using code and some other components. We tackled a lot of problems but as we figured out each problem, I learnt something new. I learnt the compexities faced while trying to link software to hardware. I also learnt how tedious it is to manually join LEDs is through wires hoping that no connection is weak. I also realized that it is important to use the correct material for every part so that every part can perfrom its task without the lamp falling apart. All in all it was a great experience which helped me put my theoretical knowledge to actual practical use.
